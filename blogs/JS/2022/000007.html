# 数组的创建

  • 字面量创建
    • var arr = [1,2,3];
  • 实例化创建
    • var arr = new Array(1,2,3);
    • 实例创建时 只有一个数字参数时 代表数组的长度

# 数组方法

# 数组的添加和删除

  • push(任意个数据项): 在数组的尾部添加任意数量个数据项，返回添加后数组的长度（方法的返回值）
  • pop()：在数组的尾部 删除一个数据项，并返回删除的该数据项
  • unshift(任意个数据项)：在数组的头部添加任意数量个数据项，返回添加后数组的长度（方法的返回值）
  • shifit()：在数组的头部 删除一个数据项，并返回删除的该数据项

# 数组的添加 删除 替换

  • splice(startIndex起始下标， deleteCount删除的个数，一组要添加的数据)
    • 从起始下标开始向后删除count个数据项，也可以添加n个数据项。有返回值 返回一个由删除的数据项组成的新数组。会改变原数组。

# 数组其他方法

  • join('标识符') 拼接为字符串
  • arr.concat(一组数据项或数组) 合并数组 返回为新数组
  • reverse() 反转数组的排列 改变原数组
  • indexOf(数据项) lastIndexOf(数据项)
    • 查找数组中是否包含该数据项 如果包含返回该数据项下标 不包含 返回-1
    • lastIndexOf 从后向前查找
  • slice(起始下标， 结束下标) 从数组中获取（不改变原数组） 起始不包含结束下标 该范围的数据项 返回为一新数组
  • Array.isArray() 判断是否为数组

# 数组的迭代方法

  • arr.every() 遍历数组, 所有数据项都满足条件表达式 返回 true/false
  • arr.some() 遍历数组, 有一项满足条件表达式 返回 true/false
  • arr.filter() 过滤数组 返回为一个新的数组
  • arr.map() 遍历数组 返回为一个新数组
  • arr.forEach（） 遍历数组， 无返回值

# 数组对象自带的排序方法

  • sort（function(a, b){return a - b / return b - a}）
    • 从小到大 return a - b
    • 从大到小 return b - a
    • 如果 相减的值 为正数 就将 a 和 b 进行互换位置 如果 相减值为 负数 则不交换位置
// sort
var arr = [23,4,56,12,25,34,7,66,42,85,6,45,33];
console.log(arr.sort(function(a, b){
    // 从小到大
    // return a - b;
    // 从大到小
    return b - a;
}));

  • sort排序数组对象
// 按 date 进行降序排序，如果 date 一样，按 DIU 进行降序排序
var arr = [;
    {"date":"2018-08-01", "DIU": 1209, "country": "US"},
    {"date":"2018-08-02", "DIU": 680, "country": "GB"},
    {"date":"2018-08-01", "DIU": 2311, "country": "CN"},
    {"date":"2018-08-02", "DIU": 879, "country": "US"},
    {"date":"2018-08-03", "DIU": 1525, "country": "CN"},
    {"date":"2018-08-02", "DIU": 1525, "country": "CN"}
];
// 字符串日期  直接转换为 时间戳 
// console.log(Date.parse('2021-10-12 14:48:00'));
arr.sort(function(a, b){
    if(a.date == b.date){
        return b.DIU - a.DIU;
    }
    return Date.parse(b.date) - Date.parse(a.date)
})
console.log(arr);

  • sort排序中文
// 中文排序（localeCompare 以本地规则排序）
var arr = [
    { name: '武丽昕', num: 78 },
    { name: '汤文博', num: 38 },
    { name: '卢文博', num: 58 },
    { name: '邓钧键', num: 97 },
    { name: '刘继昂', num: 56 },
    { name: '栗军安', num: 78 },
    { name: '屈晓月', num: 98 },
    { name: '付秋萍', num: 79 }
];
arr.sort(function(a, b){
    return a.name.localeCompare(b.name, 'zh');
})
console.log(arr);

  • 随机排序（打乱数组）
// 随机排序
var arr = ['a','b','c','d','e','f','g','h'];
arr.sort(function(){
    return Math.random() - 0.5;
})
// (0-0.99999) - 0.5
console.log(arr);
