# 选择器

# 属性选择器

  1. E[attr^="value"]：该选择器选择属性名为 attr ，属性值以 value 开头的元素。
  2. E[attr$="value"]：该选择器选择属性名为 attr，属性值以 value 结尾的元素。
  3. E[attr*="value"]：该选择器选择属性名为 attr，属性值包含 value 的元素。

# 伪类选择器

  1. E:root：该选择器选择文档的根节点，效果和设置 html 相同。
  2. E:first-child：父元素的第一个子元素。
  3. E:last-child：父元素的最后一个子元素。
  4. E:only-child：父元素唯一的子元素。
  5. E:only-of-type：父元素唯一的子元素类型。
  6. E:nth-child(n)：父元素的第 n 个子元素。
  7. E:nth-last-child(n)：父元素的倒数第 n 个子元素。
  8. E:nth-of-type(n)：父元素定义类型的第 n 个子元素。
  9. E:nth-last-of-type(n)：父元素定义类型的倒数第 n 个子元素。
  10. E:first-of-type：该选择器选择 E 元素父元素的子元素的第一个与 E 元素类型一致的元素。
  11. E:last-of-type：该选择器选择 E 元素父元素的子元素的最后一个与 E 元素类型一致的元素。
  12. E:focus 处于获得焦点状态下的元素
  13. E:default 默认元素
  14. E:valid(:invalid) 根据输入数据验证， 匹配有些（无效）的input元素
  15. E:in-range(out-of-range) 匹配在置顶范围之内（之外）受限的input元素
  16. E:empty：该选择器选择的是空元素。
  17. E:target：该选择器选择的是一个 id 与 当前 url 匹配的元素。
  18. E:not(s)：该选择器匹配不符合参数选择器 s 描述的元素。
  19. E:enabled & E:disabled：该选择器匹配的是元素的可用和禁用状态。
  20. E:checked：该选择器一般用于 radio-button 或 checkbox，选择它们的 checked 状态。

# 链接伪类

  1. E:link：未访问时的状态
  2. E:visited：访问后的状态
  3. E:hover：鼠标经过、悬停时的状态
  4. E:active：鼠标点击不动时的状态

# 伪元素选择器

  1. E::before 用于在被选元素的内容之前插入内容
  2. E::after 用于在被选元素的内容之后插入内容

# 圆角边框

border-radius

  • border-top-left-radius
  • border-top-right-radius
  • border-bottom-right-radius
  • border-bottom-left-radius

# 阴影

  1. box-shadow 盒子阴影
    • x 偏移量 | y 偏移量 | 阴影模糊半径 | 阴影扩散半径 | 阴影颜色
  2. text-shadow 文字阴影
    • offset-x | offset-y | blur-radius | color

# 背景&渐变

# 背景

  1. background-color 背景颜色
  2. background-repeat 背景是否重复
    • epeat 默认值，背景会沿着x轴 y轴方向重复
    • repeat-x 沿着x轴方向重复
    • repeat-y 沿着y轴方向重复
    • no-repeat 背景图片不重复
  3. background-position 背景图片的位置
    • top、left、right、bottom、center 几个表示方位的词来设置背景图片
    • x 水平方向的偏移量 y 垂直方向的偏移量
  4. background-image 背景图片
  5. background-size 背景图像的大小
    • width height 如果只写一个，则第二个值默认是 auto
    • cover 图片的比例不变，将元素铺满
    • contain 图片的比例不变，将图片再元素中完整显示
  6. background-origin 背景图像的位置区域
    • padding-box 默认值，background-position从内边距处开始计算
    • border-box 背景图片的偏移量从内容区处计算
    • content-box 背景图片的变量从边框处开始计算
  7. background-clip 背景剪裁
    • border-box 默认值，背景会出现在边框的下边
    • padding-box 背景不会出现在边框，只出现在内容区和内边距
    • content-box 背景只会出现在内容区
  8. background-attachment 背景图片是否随元素移动
    • scroll 默认值 背景图片会跟随元素移动
    • fixed 背景图片会固定再页面中，不会随元素移动

# 盒子模型

box-sizing

  • content-box 默认值。如果你设置一个元素的宽为 100px，那么这个元素的内容区会有 100px 宽，并且任何边框和内边距的宽度都会被增加到最后绘制出来的元素宽度中。
  • border-box 要设置的边框和内边距的值是包含在 width 内的。

# 过渡动画

transition

  • transition-property ：规定设置过渡效果的css属性名称
  • transition-duration ：规定完成过渡效果需要多少秒或毫秒
  • transition-timing-function ：指定过渡函数，规定速度效果的速度曲线
    • linear | ease | ease-in | ease-out | ease-in-out | cubic-bezier(n,n,n,n);
  • ransition-delay ：指定开始出现的延迟时间 默认值为：all 0 ease 0

# 帧动画

animation

  1. 利用@keyframes声明一个关键帧租.
  2. 在animation属性中调用上述声明的关键帧租,实现动画 animation属性的值：
  • animation-name: 指定一个关键帧动画的名字，由@keyframes声明
  • animation-duration: 动画持续时间，单位为s或ms
  • animation-timing-function: 速度曲线：
    • ease 默认, 慢-快-慢
    • linear 匀速
    • ease-in 低速开始, 加速
    • ease-out 低速结束, 减速
    • ease-in-out 以低速开始和结束 ，先加速后减速
    • steps() 指定了时间函数中的间隔数量（步长）
  • animation-delay: 动画开始之前的延迟时间
  • animation-iteration-count: 指定动画的循环次数
    • infinite 无限循环
  • animation-direction: 动画的播放方向 来回播放
    • normal 不倒回
    • alternate 逆播放
  • animation-fill-mode: 动画的时间外属性 动画结束状态
    • backwards 回到起始
    • forwards 保持
  • animation-play-state: 动画的播放状态 鼠标放上运行或暂停
    • running 不暂停
    • pause 鼠标放上暂停

# 2D转换

transform

  • rotate 旋转
    • rotate() 给定度数顺时针旋转的元素。负值表示元素按逆时针旋转。单位deg
  • scale 缩放
    • scale(x, y) 宽度（X轴）和高度（Y轴）
  • translate 平移
    • tarnslate(x, y) 根据左（X轴）和顶部（Y轴）位置给定的参数，从当前元素位置移动。
  • skew 倾斜
    • skew(x, y) X轴和Y轴倾斜的角度，如果第二个参数为空，则默认为0，参数为负表示向相反的方向倾斜 单位deg

transform-origin 变形的基准点

  • 水平位置：left、center、right
  • 垂直位置：top、center、bottom

# 3D变换

transform-style：preserve-3d transform

  • 旋转 分别使用rotateX方法、rotateY方法、rotateZ方法使元素围绕X轴、Y轴、Z轴旋转。
  • 缩放 分别使用scaleX方法、scaleY方法、scaleZ方法使元素按X轴、Y轴、Z轴进行缩放。
  • 移动 分别使用translateX方法、translateY方法、translateZ方法、使元素在X轴、Y轴、Z轴方向上进行移动。
  • 倾斜 分别使用skewX方法、skewY方法使元素在X轴、Y轴上进行顺时针方向倾斜（无skewZ方法）。

# 弹性盒子

# 父元素属性

  1. display
    • flex：块级的弹性盒子
    • inline-flex：行级的弹性盒子
    • -webkit-flex：苹果和谷歌的兼容性
  2. flex-direction:子元素在父容器中的位置
    • row：横向从左到右排列(左对齐)，默认
    • row-reverse：横向右对齐元素反转
    • column：纵向排列
    • column-reverse：反转纵向排列
  3. flex-wrap:子元素换行方式
    • nowrap：默认， 弹性容器为单行
    • wrap：弹性容器为多行
    • wrap-reverse：反转 wrap 排列
  4. flex-flow:flex-direction 和 flex-wrap复合写法
  5. align-items 基于交叉轴对齐（单行）
    • flex-start：弹性盒子元素的侧轴（纵轴）起始位置的边界紧靠住该行的侧轴起始边界
    • flex-end：弹性盒子元素的侧轴（纵轴）起始位置的边界紧靠住该行的侧轴结束边界
    • center：弹性盒子元素在该行的侧轴（纵轴）上居中放置
    • baseline：如弹性盒子元素的行内轴与侧轴为同一条，则该值与’flex-start’等效。其它情况下，该值将参与基线对齐
    • stretch：如果指定侧轴大小的属性值为’auto’，则其值会使项目的边距盒的尺寸尽可能接近所在行的尺寸，但同时会遵照’min/max-width/height’属性的限制
  6. align-content:基于交叉轴对齐（多行）
    • stretch：默认。各行将会伸展以占用剩余的空间。
    • flex-start：各行向弹性盒容器的起始位置堆叠。
    • flex-end：各行向弹性盒容器的结束位置堆叠。
    • center：各行向弹性盒容器的中间位置堆叠。
    • space-between：各行在弹性盒容器中平均分布。
    • space-around：各行在弹性盒容器中平均分布，两端保留子元素与子元素之间间距大小的一半。
  7. justify-content：基于主轴对齐
    • flex-start：弹性项目向行头紧挨着填充. 默认
    • lex-end：弹性项目向行尾紧挨着填充
    • center：弹性项目居中紧挨着填充
    • space-between：弹性项目平均分布在该行上
    • space-around：弹性项目平均分布在该行上，两边留有一半的间隔空间

# 子元素属性

  1. flex-grow：拉伸(默认0)
  2. flex-shrink：收缩(默认1)
  3. flex-basis：宽度(默认0)
  4. flex：复合写法
  5. order：定义排列顺序(默认0)
  6. align-self：子元素自身在侧轴（纵轴）方向上的对齐方式
    • auto：默认,如果’align-self’的值为’auto’，则其计算值为元素的父元素的’align-items’值，如果其没有父元素，则计算值为’stretch’。
    • flex-start：弹性盒子元素的侧轴（纵轴）起始位置的边界紧靠住该行的侧轴起始边界。
    • flex-end：弹性盒子元素的侧轴（纵轴）起始位置的边界紧靠住该行的侧轴结束边界。
    • center：弹性盒子元素在该行的侧轴（纵轴）上居中放置。（如果该行的尺寸小于弹性盒子元素的尺寸，则会向两个方向溢出相同的长度）。
    • baseline：如弹性盒子元素的行内轴与侧轴为同一条，则该值与’flex-start’等效。其它情况下，该值将参与基线对齐。
    • stretch：如果指定侧轴大小的属性值为’auto’，则其值会使项目的边距盒的尺寸尽可能接近所在行的尺寸，但同时会遵照’min/max-width/height’属性的限制。

# 网格布局

  1. display:
    • grid 指定容器使用网格布局
    • inline-grid 指定容器设为行内元素并使用网格布局
  2. grid-template-columns 定义每一列的列宽 grid-template-rows 定义每一行的行高
    • repeat(次数,值) 重复赋值 repeat(auto-fill,30px)自动填充
    • fr 比例 如 grid-template-columns: 1fr 2fr;意思是容器分为2列，二列的宽度是一列的2倍
    • minmax() 长度范围
    • auto 自适应
    • [xx,xx,..] 网格线名称
  3. grid-gap: x y ;设置上下左右外边距；
  4. row-gap:行间距; colum-gap:列间距; gap简写形式
  5. grid-template-areas 网格布局允许指定"区域"（area）
  6. grid-auto-flow 划分网格以后，容器的子元素会按照顺序 默认是row
  7. justify-items：属性设置单元格内容的水平位置（左中右）; align-items：属性设置单元格内容的垂直位置（上中下）;
    • start：对齐单元格的起始边缘。
    • end：对齐单元格的结束边缘。
    • center：单元格内部居中。
    • stretch：拉伸，占满单元格的整个宽度（默认值）。
  8. justify-content：整个内容区域在容器里面的水平位置（左中右）; align-content：整个内容区域的垂直位置（上中下）;
    • place-content：align-content justify-content; 简写

# 多列布局

  1. columns 对象的列数和每列的宽度。复合属性 集成column-width和column-count两个属性
    • column-width定义每列列宽度
    • column-count定义分分列列数
  2. column-gap 用于设置列与列之间的间距，该属性需要为多列显示时的元素设置
  3. column-rule 用于定义列与列之间的边框，其中包括边框宽度，边框颜色以及边框样式
    • column-rule-width：用于表示列与列之间的边框宽度
    • column-rule-color：用于表示列与列之间的边框颜色
    • column-rule-style：用于表示列与列之间的边框样式
  4. column-span 用于定义一个列元素是否跨列
    • none：用于表示元素不跨列
    • all：用于表示元素跨所有列
  5. column-fill 定义列的高度是由内容决定，还是统一高度
    • auto：默认值。用于表示列的高度由内容决定
    • balance：用于表示列的高度根据内容最多的一列高度为准
