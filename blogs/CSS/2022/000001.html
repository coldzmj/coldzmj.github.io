# 一 CSS3 有哪些新特性

  • 媒体查询（可以查询设备都物理像素然后自行去适应操作）
  • transform，transition，skelw，rotate等相关动画效果
  • box-shadow，text-shoadw等特效
  • @font-face规则（设计师可以引入任意的字体了）
  • @keyframes规则（可以自己创建一些动画等）
  • 2D，3D转换
  • 添加了border-radius，border-image等属性
  • 创建多列（colum-count规定文本可以以几列的方式布局）
  • 用户界面（resize，box-sizing， outline-offset）

# 二 如何让一个元素居中

  • 父元素高度固定，子元素设置margin:auto
  • 父元素高度固定，父元素设置text-align:centger
  • 水平垂直居中:
    • 子相父绝定位 子元素margin:auto
    • 子相父绝定位 子元素,上,左设置50%,子元素margin-top和margin-left减去各自宽高的一半
    • 子元素设置translate(x, y)
    • 父元素设置弹性盒display:flex
div {
    display:flex;
    justify-content:center;
    align-items:center;

# 四 请简述弹性盒布局属性

  • 原理:给父盒子添加dispaly:flex
  • flex-direction:弹性容器中子元素排列方式(主轴排列方式)
  • flex-wrap:设置弹性盒子的子元素超出父容器时是否换行
  • flex-flow:是flex-directions和flex-wrap简写形式
  • align-items:设置弹性盒子元素在侧轴的对齐方式
  • justilfy-content:设置弹性盒子再来元素主轴上的对齐方式

# 五 简述px和em和rem

  • px是像素单位
  • rpx是为了适配不同屏幕相对单位,把不同的屏幕都等分成750份
  • em是相对父级单位(如父级的div是20px,那么子元素1em等于20px)
  • rem是相对与根级的单位(如html设置为20px那么其他元素1rem等于20px)

# 六 vh和vw的理解

vh相对于屏幕的高度,相对值 vw相对于屏幕的宽度,相对值

# 八 定位属性有什么区别

  • position有四个属性值：relative absolute fixed static
  • relative:相对定位 不脱离文档流 相对于自身定位
  • absolute:绝对定位 脱离文档流 相对于开启定位的上级元素
  • fixed:固定定位 脱离文档流 相对于浏览器窗口定位
  • static:静态定位 默认值 元素出现在正常流中

# 十 弹性盒

  • 容器属性

    1. 声明弹性盒
      • display:flex
      • display:inline-flex;
    2. 设置主轴方向(默认横轴)
      • flex-direction:
        • row: 横轴(从左到右)
        • row-reverse: 横轴(从右到左)
        • colum: 竖轴(从上到下)
        • colum-reverse: 竖轴(从下到上)
    3. 设置主轴的排列方式
      • justify-content:
        • flex-start: 行头排列 默认值
        • flex-end: 行尾排列
        • center: 居中显示
        • space-between: 两端对齐,中间自动分配
        • space-around: 每个项目左右间隔一致
        • space-evenly: 每个项目的间隔一致
    4. 设置辅轴的排列方式
      • align-items：
        • row 默认值 高度铺满父元素
        • flex-start 最顶部
        • flex-end 最底部
        • center 垂直居中显示
        • baseline 基线对齐
    5. 多行排列方式
      • align-content:
        • stretch 默认值 各行将会伸展以占剩余的空间
        • flex-start 各行向弹性盒容器的起始位置堆叠
        • flex-end 各行向弹性盒的结束位置堆叠
        • center 中间位置堆叠
        • space-around 各行平均分布
        • space-between 各行平均分布，两端保留元素与子元素之间间距大小的一半
    6. 是否换行（默认收缩显示 不换行）
      • flex-warp
        • nowrap 不换行 默认值
        • wrap 换行

  • 项目属性

    1. 定义项目顺序 order：n
    2. 设置单个项目在交叉轴的排列
      • align-self:
        • flex-start
        • flex-end
        • center
    3. 项目放大(子元素总宽度小于父元素时放大)
      • flex-grow:n; 默认值0
    4. 项目缩小(子元素总宽度小于父元素时缩小)
      • flex-shrink:n; 默认值1
